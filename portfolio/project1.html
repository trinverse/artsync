Clout Aesthetics

Reputation Management Platform for Aesthetic Medicine

Clout is a full-stack SaaS platform that helps aesthetic medical professionals build, track, and showcase their professional reputation through aggregated reviews, AI-powered analysis, and a proprietary scoring system.

The Problem

Aesthetic providers have reviews scattered across Google, Yelp, and RealSelf with no unified way to measure or improve their online reputation. Manual tracking is tedious and provides no actionable insights.

The Solution

Clout aggregates multi-platform reviews, runs AI-powered sentiment analysis on each one, and distills everything into a single CLOUT Score â€” a proprietary reputation metric weighted across endorsements (40%), reviews (30%), certifications (20%), and engagement (10%).

Key Features

Multi-Platform Review Aggregation Automated collection from Google Reviews, Yelp, and RealSelf with hash-based deduplication.
AI Sentiment Analysis OpenAI-powered analysis extracting sentiment scores, topic categories, aspect-based ratings, and emotional tone.
Proprietary CLOUT Score Multi-factor reputation scoring with historical tracking and logarithmic scaling.
Peer Endorsement System Provider-to-provider endorsements with relationship tracking and quality weighting.
Analytics Dashboard Interactive charts, sentiment trends, competitive benchmarking, and exportable reports.
Real-Time Updates WebSocket-powered live feeds for review notifications and score changes.

Tech Stack

Frontend

React, TypeScript, Tailwind CSS, Recharts, React Query

Backend

FastAPI (Python), SQLAlchemy 2.0, PostgreSQL 15, Alembic

AI/ML

OpenAI GPT-4o-mini with specialty-aware prompts

Infrastructure

Docker, Nginx, GCP Cloud Run, Firebase Hosting

Architecture Highlights

  • 8-service Docker production stack: PostgreSQL, Redis, FastAPI, Celery Worker, Celery Beat, Flower, React, Nginx.
  • Dual authentication system: Local passwords and Firebase social auth.
  • Scheduled background processing: Daily review collection, score recalculation, and weekly cleanup.
